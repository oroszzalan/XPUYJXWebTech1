Alap Git & GitHub parancsok

Konfiguráció

        git config --global user.name  Felhasználói név beállitása ez a commithoz kapcsolódik
        git config --global user.email Beállizja a Git email címed ez a commithoz kapcsolódik
        git config --list Kilistázza az összes Git konfigurációt

Új repo létrehozása

        mkdir projekt-nev Új mappa a projekthez
        cd projekt-nev Belépés a mappába
        git init Git repo inicializálása
        git commit -m "üzenet" Commit létrehozása üzenettel

Klónozás

        git clone https://github.com/felhasznalo/repo.git
        git clone git@github.com:felhasznalo/repo.git

        meglévő repo másolása saját gépre HTTPS vagy SSH alapján

Változások kezelése

        git status megmutatja a változatott fájlokat a staging areaba
        git add file.txt hozzáad egy file nevű txt a staging areaba

Push & Pull

        git remote add origin URL kapcsolat egy másik GitHUb repoval
        git push -u origin main Feltölti a commitokat a main ágra
        git pull origin main Lehúzza a Githubon lévő változásokat
